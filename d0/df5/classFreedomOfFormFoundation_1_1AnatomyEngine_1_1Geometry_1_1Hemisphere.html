Construct a new Hemisphere at the point center, pointing in the direction of direction. The radius is defined by a two-dimensional function radius.

The surface is parametrized with the coordinates \(u \in [0, 2\pi]\) (the azimuthal angle) and \(v \in [0, \frac{1}{2} \pi]\) (the inclination angle).

Parameters
centerThe position of the hemisphere.
directionThe vector that defines which way is 'forward' for the object. Will be normalized on initialization.

Construct a new Hemisphere at the point center, pointing in the direction of direction. The radius is defined by a two-dimensional function radius.

The surface is parametrized with the coordinates \(u \in [0, 2\pi]\) (the azimuthal angle) and \(v \in [0, \frac{1}{2} \pi]\) (the inclination angle).

Parameters
centerThe position of the hemisphere.
directionThe vector that defines which way is 'forward' for the object. Will be normalized on initialization.
normalThe vector that defines which way is 'up' for the object. Should be perpendicular to direction and binormal. Will be normalized on initialization.
binormalThe vector that defines which way is 'left' for the object. Should be perpendicular to direction and normal. Will be normalized on initialization.
